# raspicam-server

## Requirements.
- Raspbery pi hardware. over than `model 4B`.
- Programing modules.
    - Python 3.11 over.
- External devices.
    - Microphone connect by usb cable.
    - Camera module connect by FFC cable.

## First step.
1. Create venv.
``` bash
$ python -m venv .venv
```

1. Activate venv.
``` bash
$ source ./.venv/bin/actvate
```

1. Install libraries.
``` bash
$ pip install -r requirements.txt
```
